The **Command Center** at the bottom has three sections:

- **Left** — Agent identity (name, type, state)
- **Center** — Tabs for Info (metrics), Logs (event stream), Medals (achievements), and Skills
- **Right** — Control grid: Pause, Isolate, Center, Connect, Spawn, Export, Screenshot, Marketplace, Demo, Info

Select a planet to see its live metrics: load, tool calls, prompts, errors, uptime, and an activity sparkline.
