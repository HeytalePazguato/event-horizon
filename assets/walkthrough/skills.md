Event Horizon discovers your installed **Agent Skills** (`SKILL.md` files) and shows them in the **Skills tab**.

- **Browse** skills with scope, agent type, and category badges
- **Create** new skills with the guided wizard (click **+** in the Skills tab)
- **Organize** skills into category folders (Move action on skill cards)
- **Marketplace** — browse skill marketplaces to find and install new skills (click Marketplace in the control grid)

Each planet shows a **skill orbit ring** — dots representing compatible skills that pulse cyan when a skill is actively executing.
